## Install
### 개발 서버
```sh
./dev_server.sh
```
### 운용 서버
```sh
./production_server.sh
```

### .env 셋팅
frontend 폴더 아래에 .env 파일을 만듭니다. 파일 생성 후 다음과 같이 코드를 추가해줍니다.
```
NODE_PATH=src/
REACT_APP_HOST=https://example:433
REACT_APP_UPLOAD=https://example:433
```
